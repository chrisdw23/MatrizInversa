MatrizInversa

¿Qué hace?
Este programa calcula la matriz inversa de una matriz cuadrada.

¿Cómo funciona?
- Se aplica el método de Gauss-Jordan:
  - Se convierte la matriz original en la matriz identidad.
  - Al mismo tiempo, se transforma la identidad en la inversa.
- Si la matriz no es invertible (determinante = 0), el programa lo indica.

Ejemplo
Entrada: 
 2 1 1
 1 3 2
 1 0 0

 y la salida quedaria como: 
 0.0000 0.0000 1.0000 
 -2.0000 1.0000 3.0000
3.0000 -1.0000 -5.0000
